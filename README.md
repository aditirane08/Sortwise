# Sortwise 🗑️

Sortwise is a small project I made to help with waste segregation. The idea is simple — you upload an image of an item, and the system tells you which bin it should go into.

## 🚀 What it does

- Takes an image as input
- Uses AI logic to understand the object
- Suggests the correct bin

The bins included are:
- 🔴 Red (hazardous waste)
- 🔵 Blue (dry waste)
- 🟢 Green (wet waste)
- 🟡 Yellow (recyclables)
- ⚫ Black (general waste)

## 💡 Why I made this

Most people don’t really know where to throw certain waste. I wanted to make something that makes this easier and reduces wrong disposal.

## 🛠️ Tech used

- HTML, CSS, JavaScript
- Basic AI logic (image-based sorting)

## 📂 How to use

1. Open the project in your browser
2. Upload an image
3. Wait for the result
4. It will tell you the correct bin
